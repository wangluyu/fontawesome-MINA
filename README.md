# fontawesome-MINA
小程序使用fontawesome

### 如何使用
- 拷贝x.x.x文件夹(例如5.8.1/)到小程序根目录
- 在app.wxss引入x.x.x文件夹下所有文件，例如
  ```
  @import "fontawesome-brands";
  @import "fontawesome-regular.wsxx";
  @import "fontawesome-solid.wsxx";
  @import "fontawesome.wsxx";
  ```
- 在wxml中添加代码 ```<i class="fab fa-github"></i>```
